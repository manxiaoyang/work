
# 作品背景
## 现在大家的生活变得越来越好，养宠物的人变得越来越多，但是随之而来的也产生了一些问题，比如流浪宠物大大增加，自己家的宠物生产一下好几只不知如何处理等问题，所以基于以上问题开发了一个宠物领养中心，在这里大家可以发布自己的领养信息，其他人可以通过信息联系到宠物的主人，二人可以协商如何对宠物进行领养。

# 作品简介
## 本作品开发采用了Moralis和Filecoin/IPFS等技术快速实现了一个去中心化的app，网页管理开发运用了Moralis技术，信息存储采用Filecoin/IPFS技术，智能合约部署到测试网上
## 用户可以通过设置页面来设置自己的头像、背景图以及其他个人信息，通过发布页面来发布自己的领养需求，平台的所有人都可以看到所有用户发布的领养信息。

## 运行作品所需的必要内容:
[MetaMask钱包插件下载链接](https://metamask.io/)
[在Moralis官网上申请自己的APP_ID 和serverURL](https://admin.moralis.io/dapps)

## 启动指南：
#### 1.npm install （安装依赖）
#### 2.修改index.js中的appId 和 serverUrl 改为自己在Moralis官网申请的
#### 3.修改truffle-config.js钱包助记词
#### 4.npm run start 启动项目
